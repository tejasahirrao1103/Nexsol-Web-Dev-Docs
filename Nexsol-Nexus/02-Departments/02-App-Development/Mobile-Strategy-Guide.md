# Nexsol App Development: Mobile Strategy (Internal Playbook)

**Target Audience:** Mobile Developers & UI/UX Designers.
**Objective:** Building seamless, high-performance mobile ecosystems for SMEs.

---

## 1. The Mobile Advantage
Indian SMEs thrive on mobile-first interaction. We don't just "wrap" websites; we build performance-native experiences.

---

## 2. Tech Stack Standard
- **Core:** React Native / Flutter (Client dependent).
- **Backend:** Node.js / Supabase.
- **Push Engine:** Firebase Cloud Messaging (FCM).
- **Offline Mode:** Priority for semi-urban users.

---

## 3. The "Storefront-in-Pocket" SOW
Deliverables for Every App Project:
- [ ] Cross-Platform build (iOS & Android).
- [ ] Real-time inventory sync with ONDC.
- [ ] One-Tap checkout with UPI integration.
- [ ] AI-driven product notifications.

---

## 4. Engineering Standards
- **Startup Time:** Must be < 3 seconds.
- **Bundle Size:** Optimized to < 20MB for low-storage devices.
- **Security:** Fingerprint/FaceID login for enterprise apps.

---

## 5. Handover & Publishing SOP
1. **Testing:** FlightTest (iOS) and Internal Testing (Android).
2. **Review:** Comprehensive manual check for "Thumb-Friendly" UI.
3. **Submission:** 5-day lead time for App Store / Play Store reviews.
